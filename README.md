# Chatter

## Stack

- NodeJs
- expressJS
- vanilla script
- socket io
- qs
- moment

## Usage

```
npm install
npm run dev

Go to localhost:3000
```

## Notes

The _\_html_css_ folder is just a starter template to follow along with the tutorial at https://www.youtube.com/watch?v=jD7FnbI76Hg&t=1339s. It is not part of the app
